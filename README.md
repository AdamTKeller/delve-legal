# delve-legal

Legal documents for the [Delve](https://github.com/AdamTKeller) mobile app.

Hosted publicly via GitHub Pages so the app stores have a stable HTTPS URL to link to.

## Documents

- [`index.md`](./index.md) — Privacy Policy

## Editing

The privacy policy is a single markdown file. Edit `index.md`, bump the **Last updated** date, commit, and push — GitHub Pages will redeploy within a minute.

## Live URL

After Pages is enabled (Settings → Pages → main branch, root): `https://adamtkeller.github.io/delve-legal/`
